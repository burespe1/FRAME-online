Site navigation
 Tabs  Navigator  Zoom  

Principle

Site pages use a tab mechanism enabling categorization of information.
Clicking a tab selects and displays its content.

Example

This page presents 3 tabs :
  - Tabs (currently selected)
  - Navigator
  - Zoom

The current tab is "Tabs" whichs explains the tab mechanism.

Other information

- When a page displays diagrams, clicking on the "Diagrams" tab diplays the list of available diagrams.

- "Tabulation"/"Shift+Tabulation" keys change the current tab.

Principle

The left part of the site is a navigator presenting site objects as a tree.
Clicking an object in the tree displays the description of this object in the right part of the site.

Changing the navigator tree

Clicking one of the buttons above the tree changes the displayed tree :
  - Selects overviews tree
  - Selects business functions tree
  - Selects business processes tree
  - Selects procedures tree
  - Selects org-units tree

Other information

- closes and opens the navigator.

- finds in the tree, the object displayed in the right part of the site.

- "Up arrow"/"Down arrow"/"Left arrow"/"Right arrow" keys enables movement in the tree.

Zoom

When the right part of the site displays a diagram, diagram size can be changed :
  - ajusts diagram width to fit the screen.
  - returns diagram to actual size.
  - enlarges diagram size.
  - reduces diagram size.
