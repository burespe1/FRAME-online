Dataflow
mt.ptja_road_network_characteristics


Description

It embodies the structure of the road network. This includes information for trip planning for all kinds of vehicles: restrictions for certain freight (e.g. hazardous goods) or vehicles (e.g. weight, dimension, etc.), access restrictions (e.g. one direction), but also travel times between locations in the network. Extra access rights for emergency vehicles (e.g. ambulance) need to be incorporated.
