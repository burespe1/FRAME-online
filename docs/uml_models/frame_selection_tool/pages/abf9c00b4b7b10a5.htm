Low Level Function
3.1.1.11 Provide Updated Urban data for Digital Maps

Overview

This Function shall be capable of providing the following facilities:

(1) The provision of up-to-date information for digital maps and related databases for segments of the urban road network managed by the System to the Geographic Information Provider.
(2) The information shall include such things as structural alterations, static speed limits and default journey times.
(3) The information shall be provided in a way that it can be used by the Geographic Information Provider in subsequent issues of digital maps and any other related data they provide for use by in-vehicle devices.

Functional Requirements

(a) continuously monitor for receipt of the input data flows
(b) when any of the data flows in (a) is received, apply any necessary processing so that the data is suitable for use by a digital map provider
(c) on completion of (b), send the data to the geographic information provider in the output data flow.
