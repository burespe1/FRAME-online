Dataflow
pepf_contract_CAP


Description

  It contains all the elements required to define the contract between the user and the operator.  The data flow contains the following elements:
- user ID
- operator ID
- list of services ID covered by the contract
- area covered
- supplementary parameters defining the service
- period of validity
- tariffs (optional)
- reduction fees (optional)
- modes of payment
- EP account ID (optional)
