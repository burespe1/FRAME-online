Dataflow
tcc.fs-fleet_transport_opportunity_request


Description

It is a request from the fleet supplier to Freight Shipper to get information about current freight opportunity.  The data flow includes Fleet Supplier name and address, plus criteria for transport opportunity selection, i.e. start time, end-time, start location, end location and cargo type.
