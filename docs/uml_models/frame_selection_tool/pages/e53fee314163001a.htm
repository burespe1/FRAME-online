Low Level Function
6.5.3.3 Collect PT Data

Overview

This Function shall be capable of providing the following facilities:

(1) The ability to collect Public Transport travel data for use in the preparation of trip plans for Travellers.
(2) The ability to collect the data as it arrives from functionality in the Manage Public Transport Operations Functional Area.
(3) The ability to load the data that has been collected into the store of PT Trip Planning Data and to manage all of the data in that store.

Functional Requirements

(a) the Function shall collect data as it arrives from functionality in the Manage Public Transport Operations Functional Area
(b) the Function shall load the data into the store of PT Trip Planning Data as soon as it is received
