Description

mffo.psle_transfer_data Trigger Flow: No
This data flow is sent by the Manage Freight and Fleet Operations Area to the Provide Support for Law Enforcement Area. It carries all information from Manage Freight and Fleet Operation area to Provide Support for Law Enforcement in order to characterise fraud if any. The data flow consists of the following items each of which has its own data flow definition:
mffo.psle_fraud_data
+ mffo.psle_fraud_notification
