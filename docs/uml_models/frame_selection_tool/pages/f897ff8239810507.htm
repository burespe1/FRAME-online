Overview

This High-level Function shall assess the possibilities for inter-modal transport activity, where the assessment is related to transport infrastructures.  It shall optimise the use of transport modes with respect to mobility, security and environment constraints, and shall enable freight storage areas to be reserved.  The Function shall consist of the following lower level Functions:
