Dataflow
pshvs_road_curve_ahead_warning


Description

It contains data from which a warning for the Driver of the Host Vehicle can be generated containing advice on the action(s), e.g. changes to Vehicle speed and/or trajectory, which need to be taken to ensure that the Vehicle travels safely through the curve in the road network that is ahead of the Vehicle in its current trajectory.
