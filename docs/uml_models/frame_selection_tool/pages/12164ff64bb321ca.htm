pshvs.mt_road_use_data_from_trip


Description

It contains actual journey times from the parts of a Vehicle Trip Plan that have used segments of the inter-urban and urban road networks, plus origin - destination (O-D) data and actual journey time for the complete Vehicle Trip Plan that has just been implemented.
