Dataflow
tt-pepf_overdraft_notification


Description

It is used to warn the Traveller that one of their electronic payment accounts is overdrawn, i.e. has no money for the requested payment. The data flow includes the following elements:
- date
- account ID
- user ID
- reference of last transaction
- balance before transaction
- balance after transaction
- minimal amount to credit for account re-use
