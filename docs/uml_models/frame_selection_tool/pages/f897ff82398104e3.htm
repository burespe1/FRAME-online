DFD 7.2 Identify Violator

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall analyse the image or information included in the fraud notifications sent by functionality in the other Areas, or in the images sent by the "Detect Fraud" Function, to find out the identity of the violator.  It shall use information stored in the "User Registration" database.
