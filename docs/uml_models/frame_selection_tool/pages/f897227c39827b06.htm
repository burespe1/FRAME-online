Dataflow
tmms.omfs-information_request_to_other_mode


Description

It contains details about a freight road transport to happen and requires information for being able to take account of urban delivery conditions (mobility, security and environment) as well as possible non road transport modes. This data flow includes freight operator request ID and transport conditions (freight characteristics, departure/arrival date and time, origin/destination).
