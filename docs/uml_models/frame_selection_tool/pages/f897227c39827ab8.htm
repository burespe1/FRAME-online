Dataflow
tfc-pepf_load_account_order


Description

It is used to ask the Financial Clearinghouse to transfer a certain amount from an account managed by it to an electronic payment account. The data flow includes the following elements:
- date
- user ID
- ID of the financial clearing-house
- EP account ID
- ID of the account from which the money will be transferred
- amount to be transferred
