Dataflow
td.fvd-rest_area_parking_response


Description

It contains the response to the previous request made by the Freight Vehicle Driver for a parking space at a rest zone that forms part of a service area.  It may contain either acceptance of the previously requested booking, or details of alternative times when a parking space is available, because it is not available at the requested time.
