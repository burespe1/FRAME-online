Dataflow
pepf_fraud_notification_L2


Description

It contains data that identifies a fraud performed by a user and related to electronic payment.  The fraud may either be an invalid payment, or an attempt to force one's way through the access control devices.  The data contains the following elements:
- reference
- date
- type of fraud
- result of fraud
- involved user ID (if available)
- involved vehicle ID (if relevant and available)
- image of fraud (if available)
- location of fraud
- other data according to the type of fraud
