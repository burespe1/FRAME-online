Dataflow
mt_inter-urban_traffic_management_l&s_response


Description

It contains details of the response to the previous request for displays to be shown by lane and speed output devices to drivers using the inter-urban road network.  These details may include but not be limited to the identity and/or location of the device and the actual response to the previous display command.
