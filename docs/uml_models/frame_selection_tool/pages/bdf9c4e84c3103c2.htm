Dataflow
td-road_curve_ahead_warning


Description

It contains a warning for the Driver of the Host Vehicle that it is approaching a curve in the road network and that the following action(s) should be taken, e.g. changes to Vehicle speed and/or trajectory, to ensure that the Vehicle travels safely on its current trajectory.
