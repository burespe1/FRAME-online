Dataflow
flea-psle_user_clearance


Description

It contains data that is sent by a law enforcement agency and includes parameters describing what the vehicle or user is allowed to do.  The data includes the following elements:
- user ID
- vehicle ID
- max speed according to environment conditions
- max weight (total and per axle)
- max pollution level
- type of cargo / types of forbidden zones
- max duration of continuous drive.
