Dataflow
psle_inter-urban_violator_ID


Description

It contains the identity of the driver that has just violated the traffic rules in the inter-urban road network. The data flow is sent after analysis of a fraud (violation) notification detected and sent by Functions in the Manage Traffic Area. It contains the following elements:
- fraud notification reference
- vehicle ID
- driver ID
