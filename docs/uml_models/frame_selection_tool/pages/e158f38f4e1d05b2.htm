Dataflow
mt.pshvs_inter-urban_c&i_messages


Description

It contains the contents of commands and information messages that are for In-vehicle display and use. This output will provide commands and messages for all types of Drivers to take actions so that their Vehicles will make best use of the urban road network. The location coded area where the command is to be displayed and where it is valid is also included.
