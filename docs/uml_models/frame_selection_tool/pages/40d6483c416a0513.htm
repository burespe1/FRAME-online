Dataflow
ptja_traveller_trip_description


Description

It contains the description of a trip that has been produced in response to a request from a Traveller.  Details such as the origin, destination, other places to be passed through during the trip, modes being used for each part of the trip, plus details of the required PT services and those provided by other modes.  Any changes from the trip as originally proposed by the Traveller will be highlighted.
