Data Store
7.1 Rules Data


Description of content

This Data Store shall be used within the Provide Support for Law Enforcement Area.  It shall contain all the rules and the types of fraud or violation associated with each rule.

The data in the Store for each rule shall be structured in the following way:
- rule ID (number)
- area of road network to which the rule applies ()
- description of the rule (characters)
- types of fraud (violation) that can be associated with the rule (characters)

The data in the Store for each type of fraud or violation shall be structured in the following way:
- fraud or violation ID (number)
- Fraud or violation description (characters)
- level of seriousness (characters)
- authority responsible for enforcement (characters)
- enforcement procedure (characters)
- potential sanction (characters)

It shall be possible for this data to be updated every time a change is made.
