Dataflow
pepf_fraud_history


Description

It contains the list of all frauds detected by the EP system for a given period.  The data flow contains the following elements:
- date
- type of fraud
- result of fraud
- involved user ID (if available)
- involved vehicle ID (if relevant and available)
- location of fraud
- other data according to the type of fraud
