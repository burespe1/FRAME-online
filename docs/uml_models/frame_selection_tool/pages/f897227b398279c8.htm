tcc.p-constraints_change_request


Description

It contains elements of information that relate to a proposal from the freight operator to change the road transport conditions in order to ease an inter-modal transport mode and includes principal order ID, departure/arrival dates proposal, inter-modal transport conditions and electronic signature.
