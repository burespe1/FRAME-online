Terminator Urban Traffic Management System
 Diagram   Definition    TabMemoryProcess

Overview

This Actor within the Other Related System Terminator shall represent another instance of the Urban Traffic Management System.  It shall be possible for data to be exchanged with this Actor to represent the exchange of data with an Urban Traffic Management System that covers a different geographic area.
