Dataflow
tv.ov-red_light_running_warning


Description

It contains a warning to the Other Vehicle from the Host Vehicle that it is about to run through a set of traffic signals that will be seen as red by its Driver, i.e. red light running.  The warning will include the identity of the Host Vehicle so that the Other Vehicle can determine which Vehicle is red light running if there are several Vehicles that are approaching the same traffic signals.
