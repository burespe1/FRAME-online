Data Store
5.2 ISA Data


Description of content

This Data Store shall be used within the Provide Advanced Driver Assistance Area.  It shall contain speed limit data for a region from which, given a set of co-ordinates, the speed limit for that section of road can be identified.

The data in the Store shall be structured in the following way:
- road link ID (number)
- location (characters and/or numbers, e.g. GPS/Galileo data)
- speed limit (number in kph or mph)
- recommended speed (number in kph or mph)
- weather conditions for recommended speed (characters)
