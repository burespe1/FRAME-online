Low Level Function
1.3.1 Detect User

Overview

This Function shall be capable of providing the following facilities:

(1) The ability to detect the approach of a Traveller or Vehicle.
(2) When detected, the ability to trigger the other functionality in the Provide Electronic Payment Facilities Area.

Functional Requirements

(a) detect approaching vehicle or traveller,
(b) send a detection message to the identify user Function
