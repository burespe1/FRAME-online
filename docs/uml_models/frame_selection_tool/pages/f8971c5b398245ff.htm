Dataflow
mt_tunnel_action_responses


Description

It contains responses from the Tunnel Operator to the list of automatically executed actions which have been implemented to counter exceptional conditions that have been detected in a tunnel, or new actions that are to be taken when exceptional conditions arise in the future.
