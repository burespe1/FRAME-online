Dataflow
pshvs.mt_rest_zone_use_data


Description

It contains requests and confirmations for the use of parking places in rest zones that are part of service areas, as required by the need to ensure that the journey times in a Vehicle Trip Plan conform to the European Working Time Directive for Freight Vehicle Drivers.
