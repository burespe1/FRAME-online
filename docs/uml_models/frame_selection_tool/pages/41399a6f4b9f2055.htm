Dataflow
pshvs.mt_safety_behaviour_status_for_inter-urban


Description

It contains an indication that the Vehicle is being driven in an unsafe manner through the inter-urban road network, i.e. exhibiting poor safety behaviour, so that any FCD/XFCD data it is providing can be ignored.
