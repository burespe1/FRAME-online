Dataflow
psef.pshvs_emergency_vehicle_approaching


Description

It contains a warning that an Emergency Vehicle is approaching the Host Vehicle and may require the Host Vehicle to change lanes in order to give the Emergency Vehicle an un-interrupted path through the road network, i.e. a "blue" wave.
