Low Level Function
3.1.4.8 Mange Urban Parking Data Store

Overview

This Function shall be capable of providing the following facilities:

(1) The ability to manage the use of the store of Urban Car Park Data.
(2) The ability to load the store both static and real-time data and to extract (read) this data from the store when requested by other functionality.

Functional Requirements

(a) continuously monitor for receipt of any of the input data flows
(b) when any of the data flows with actual static or real-time data is received in (a) load it into the store of Urban Car Park Data using the data flow load car park data
(c) if any of the data flows received in (a) contain a request for previously stored car park data, extract it from the store of Urban Car Park Data using the data flow, read car park data and send it to the function that requested the data.
