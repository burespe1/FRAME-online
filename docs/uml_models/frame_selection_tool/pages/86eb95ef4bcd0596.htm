Data Store
9.2 Sensitive Area Monitoring Data


Description of content

This Data Store shall be used within the Provide Support for Cooperative Systems Area.  It shall contain records of the Vehicles that have entered a "sensitive area" in the road network.

The data in the Store shall be structured in the following way:
- date (date string)
- time (numbers defining hours, minutes and seconds)
- area location (characters, e.g. nation, region, city, state, area names)
- access criteria (characters)
- vehicle ID (number)
- vehicle type (characters)
- hazardous goods on board ()
- entry point (characters and/or numbers, e.g. GPS/Galileo data)
- exit point (characters and/or numbers, e.g. GPS/Galileo data)
