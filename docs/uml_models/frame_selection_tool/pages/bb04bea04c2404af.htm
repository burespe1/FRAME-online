Dataflow
pshvs_vehicle_data_for_trajectory


Description

It contains data provided by In-vehicle Systems of the Host Vehicle that is used to calculate the forward trajectory of the Vehicle.  It therefore includes such items as the status of the power unit, driver train, front wheels, steering wheel, turn indicator and brakes, plus the Host Vehicle identity.
