This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall establish the contracts between the users of the different services and the operators providing these services. These contracts shall define the rights of the user, his or her access to the different services, and the way the user will pay for them.  The Function shall also provide the Operators with statistical data about the contracts covering their services.
