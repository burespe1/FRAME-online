Dataflow
flea-psle_violation_data_request


Description

It is used by the Law Enforcement Agencies to ask for information related to past offences. The data flow includes the following elements, some being optional:
- date of request
- date / time of fraud
- location of fraud
- violator ID
- type of fraud
