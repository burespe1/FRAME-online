Dataflow
ptja.pepf_service_data


Description

It contains information about the different services available to the user and answers a request made by the user. The data flow may include the following elements, depending on the request :
- type of service (transport, information, parking, …)
- type of information (schedule, access conditions, tariffs, …)
- location of service
- date / time of service- tariff- operator ID.
