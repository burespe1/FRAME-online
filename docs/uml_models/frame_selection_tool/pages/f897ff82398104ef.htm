DFD 7.5 Manage Rules and Users' Registrations

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall be charged with the upkeep of the Rules and Users' Registration stores.  It shall update their content with new elements issued by the law enforcement agencies.
