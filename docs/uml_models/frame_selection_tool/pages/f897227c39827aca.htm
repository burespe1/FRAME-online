Dataflow
tlea-psle_violation_notification


Description

It is sent upon receipt of a specific request to the Law Enforcement Agencies. The data it contains corresponds to the internal fraud notification, and therefore includes:
- reference
- date
- type of fraud
- result of fraud
- involved user ID (if available)
- involved vehicle ID (if relevant and available)
- image of fraud (if available)
- location of fraud
- other data according to the type of fraud
