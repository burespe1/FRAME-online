Dataflow
mt_inter-urban_incident_warning_c&i_commands


Description

It contains details of the actual information about an incident that is to be displayed to drivers and other travellers using the inter-urban road network.  Also included may be the identity and/or location of the inter-urban command and information devices through which the information is to be output.  The display of this information will override that of all other messages on the selected device(s).
