Dataflow
mt_urban_road_static_data_for_de-icing


Description

It contains the static data that defines the urban road network for use in assessing the de-icing strategies that may be required and thus it includes information such as the road geometry, numbers of lanes, junction design and the relationship between the junctions and the roads linking them together.
