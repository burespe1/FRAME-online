Dataflow
mt_service_area_occupancies_for_demand_management


Description

   It contains data about the current occupancies of service areas in the inter-urban road network that are to be used by the demand management group of Functions.    The term "service area" is generic in that it may also include locations where only freight and/or Public Transport vehicles are parked whilst in transit through the inter-urban road network.
