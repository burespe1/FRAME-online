Dataflow
to-pepf_contract_statistics


Description

It contains information about the contracts established with the Operator during a period specified in the request. The data flow includes the following elements:
- operator ID
- number of contracts for each service ID
- location of these contracts
- dates of contracts
- modes of payment chosen
- amount of contracts
- other valuable information (depending on the service nature : for example : quality of service required, duration of contract, …)
