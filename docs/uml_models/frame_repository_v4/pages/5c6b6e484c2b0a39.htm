Dataflow
pshvs_nearby_vehicles_and_road_user_information


Description

It contains information about Other Vehicles and/or Other Road Users that are nearby the Host Vehicle, including those approaching the same set of traffic signals being approached by the Host Vehicle and those approaching the front or rear of the Host Vehicle that are partially in the lane occupied by the Host Vehicle.
