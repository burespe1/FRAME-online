Dataflow
fcc.p-payment_acknowledgement


Description

It carries the acknowledgement that a payment has been placed in the bank account of the freight operator by the principal. This data flow includes freight transaction ID (principal order ID and freight operator proposal ID), date of payment and electronic signature.
