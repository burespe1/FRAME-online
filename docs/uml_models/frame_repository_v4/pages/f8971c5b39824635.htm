Dataflow
mt_urban_environmental_inputs


Description

It contains details of current and predicted environmental conditions for the geographic area managed by the System for use in managing traffic on the urban road network. This data will be used both to influence traffic management strategies and as the source of pollution warning messages for output to Drivers and Travellers.
