Dataflow
pepf_overdraft_notification


Description

It contains data that is used to warn that the EP account used to pay for a transaction does not have sufficient credit to cover the proposed transaction and that if transaction is processed the account will have (or already has) an overdraft.  The data includes the following elements:
- date
- account ID
- user ID
- reference of last transaction
- balance before transaction
- balance after transaction
