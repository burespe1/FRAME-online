Dataflow
td-pepf_payment_request


Description

It contains the request for the payment corresponding to the service ordered by the Driver.  The data flow includes the following elements:
- service ID
- other parameters defining precisely the service
- request for information about payment means :
      - mode of payment
      - account ID
- amount to be paid
