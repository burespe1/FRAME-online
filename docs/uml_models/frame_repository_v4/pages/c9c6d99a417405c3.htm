Low Level Function
5.11.6 Provide Incident Related Data

Overview

This Function shall be capable of providing the following facilities:

(1) The ability to provide the record of data relating to an incident to an authorised person at the Law Enforcement Agency.
(2) The ability to only provide this data when a request is received from that Agency.

Functional Requirements

a) When a request for incident data is received the authority of that request is checked.
b) If the request is valid the data for the period of time requested is provided to the authorised person
