Dataflow
td.tpd-request_vehicle_trip_planning_payment


Description

It contains a request to the Driver for payment for the use of the trip planning services, without which the trip plan will not become available for use.  (Note: the use of this and the corresponding "payment" data flow is optional and depends on what the trip planning service provider wants to do in a particular ITS implementation.)
