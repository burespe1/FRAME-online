Description

It is used within the Manage Freight and Fleet Operations Area. The data flow carries the information that a fleet operator has been looked for. The action can be successful or not. The data flow includes the freight transaction ID (consignor order ID and freight operator proposal ID), fleet research result (positive or not) and fleet transaction ID.
