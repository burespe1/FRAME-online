Dataflow
psle_violation_history


Description

It regroups all the frauds (violations) committed by a given user during a certain period. The data flow contains the following elements:
- user ID
- period concerned
- list of frauds committed, with:
- type of fraud
- date of fraud
- location of fraud
- fraud notification and prosecution files references
