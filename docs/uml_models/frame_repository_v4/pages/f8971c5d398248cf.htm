Dataflow
pepf_account_status


Description

It contains the balance of a user's EP account. The data flow contains the following elements:
- user_ID
- account ID
- account balance
- list of transactions performed on this account during a given period: for each transaction:
- date of transaction
- amount credited or debited
- source or destination of transaction
