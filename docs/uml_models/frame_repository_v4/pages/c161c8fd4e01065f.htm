Dataflow
pshvs.mt_urban_road_surface_state_message


Description

It contains data from which a warning message can be generated for output from the roadside to the Driver of the Host Vehicle to advise of poor and/or bad road surface conditions that will be encountered in the urban road network ahead of the Host Vehicle and any appropriate action that the Driver needs to take.
