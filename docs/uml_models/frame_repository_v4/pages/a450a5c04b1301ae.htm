Dataflow
fo.tio-inter-urban_road_network_data_update


Description

It contains static data about the inter-urban road network that is being added the Trip Planning Data Store by the Travel Information Operator for use in trip plans.  This data may be a sub-set of the static data needed for Traffic Management.
