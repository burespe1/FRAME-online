Dataflow
pshvs.mt_possible_conflicts_with_vehicle_movement


Description

It contains data from which a message can be produced advising Drivers that there are possible conflicts with Other Vehicles, Other Road Users, Pedestrians, and Vulnerable Road Users in the forward trajectory of the Host Vehicle, the identity of which is included in the data.
