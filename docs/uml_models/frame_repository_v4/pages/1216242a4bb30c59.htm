Dataflow
ptja.pshvs_vehicle_trip_plan_responses


Description

It contains responses to previous requests for the creation of Vehicle Trip Plans, either at the request of the Driver, or because the Trip Plan that is currently being implemented needs to be modified because road conditions have changed.
