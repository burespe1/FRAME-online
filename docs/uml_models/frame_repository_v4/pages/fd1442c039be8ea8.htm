User Need 9.5.1.5


Description

The system shall be able to transfer any necessary information about an operation, e.g. route, (hazardous or oversize) cargo, etc. to the relevant authorities, e.g. TCCs, TICs etc.

Functions

Here are the functions that serve this User Need
  • 8.2.2.1.1 Elaborate and Store Operational trip and load plan
  • 9.4.2 Manage and Monitor Hazardous Goods Vehicle Routes
  • 9.4.3 Provide Hazardous Goods Vehicle Driver Interface

User Need Category

9.5.1 Road Freight Management
