Dataflow
td-pepf_payment_acceptance


Description

It is sent upon verification that the received payment is correct. The data flow includes the following elements:
- date / time
- user ID
- location
- service ID
- other parameters allowing to define the service
- amount of transaction
- mode of payment (account number, time of actual debiting)
