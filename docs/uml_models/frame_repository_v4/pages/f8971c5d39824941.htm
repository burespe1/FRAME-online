Dataflow
pepf_pass_record


Description

It contains data that is used, upon user's detection, to record the fact that the user has performed a certain action at the present time.  For example, the crossing of a toll gate is recorded, so that at the next gate, the system is able to compute travel time between the two gates.  The data flow contains the following elements:
- time
- device ID
- device location
- user ID (if relevant)
- vehicle ID (if relevant)
- type of action recorded
