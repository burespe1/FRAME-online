Terminator Un/Loading Zone Operator
 Diagram   Definition    TabMemoryProcess

Overview

This Actor within the Operator Terminator shall represent a human entity that uses the facilities of the System to manage the use of Un/Loading Zones for Freight Vehicles.  The System may be in communication with more than one human entity that is a Un/Loading Zone Operator.  Each entity may belong to the same organisation that owns and/or operates loading or unloading zones, or to different organisations.
