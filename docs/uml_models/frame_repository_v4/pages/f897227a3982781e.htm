Dataflow
ft-post_trip_preferences


Description

It contains updates to the current set of Traveller's General Trip Preferences (GTP) for use in future trip planning. The data is generated as a result the success of a trip that the Traveller has just completed. Its use and availability removes from the Traveller the need for the repeated input of the same data every time a trip is planned.
