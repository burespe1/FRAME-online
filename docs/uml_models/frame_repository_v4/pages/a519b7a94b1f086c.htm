Dataflow
ptja_extra_data_for_travel_information


Description

It contains data about Points of Interest (POI), Personal Services (PS), the services provided by other transport modes and tolls (bridges, roads and tunnels) that has been gathered as part of the preparation of trip plans for Travellers.
