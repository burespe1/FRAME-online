Overview

This Actor within the Traveller Terminator represents a human entity that is currently using the facilities of the system to enable them to cross a single element that forms part of the road network.  A single element shall be a single road way that may be single or dual carriageway, with road traffic flowing in one or both directions.  The system may be in communication with more than one human entity each of which is a Pedestrian.
