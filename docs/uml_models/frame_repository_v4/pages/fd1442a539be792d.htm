User Need 7.1.11.3


Description

The system shall be able to identify those vehicles, or their drivers, which violate the parking regulations, e.g. fail to pay, stay too long, etc.

Here are the functions that serve this User Need

  • 3.1.1.5.8 Detect Urban Traffic Violations
  • 3.1.2.14.5 Detect Violations on Inter-urban Roads
