Dataflow
tesp-pepf_transaction_analysis


Description

It contains information about the transactions performed for the benefit of the External Service Provider during a period specified.  The data flow includes the following elements:
- service provider ID
- services ID
- parameters chosen by the users for these services
-corresponding amount of transactions
- location of service utilisation
- revenue generated by services managed by other information providers (revenue sharing)
- revenue distributed to other information providers (revenue sharing)
- total amount credited on each account
