Dataflow
pepf.mffo_payment_receipt_MTO


Description

It contains the payment receipts provided that are being returned to the Manage Freight and Fleet Operations Area by the electronic payment system upon reception of the payment of the transaction. It is composed of several data flows: amount of the transaction, date / time, location, mode of payment, other parameters necessary to define precisely the service provider, service ID and user ID.
