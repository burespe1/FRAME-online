Consignor/Consignee

This is a Terminator diagram which provides a description of the Terminator breakdown structure into Actors.

Overview

This Terminator shall represent human or physical entities that need freight (goods) to be transported from one place to place.  It shall consist of the following two Actors:
Freight Shipper
Principal
When the transport is being arranged, it shall be possible for the freight (goods) to be referred to as a “consignment”.
