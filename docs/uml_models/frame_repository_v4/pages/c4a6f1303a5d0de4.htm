Dataflow
tfc-pepf_account_commands


Description

It contains commands for processing by the Financial Clearinghouse. One is a request for details of the current status of a Traveller's account that is used to purchase services and the other is a request to transfer a certain amount from an account managed by it to an electronic payment account.
