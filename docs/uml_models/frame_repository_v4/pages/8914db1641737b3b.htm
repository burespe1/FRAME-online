Dataflow
fv.ov-lights_anti-glare_request


Description

It contains a request from the Other Vehicle that the beam from the headlights of the Host Vehicle which is approaching it should be lowered because they are too high and causing glare.  The request will include the identity of the Other Vehicle so that the Host Vehicle can determine which Vehicle is suffering from glare from its headlights.
