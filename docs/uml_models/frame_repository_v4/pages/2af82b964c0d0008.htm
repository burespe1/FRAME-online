Terminator Vehicle Environment
 Diagram   Definition    TabMemoryProcess

Overview

This Terminator shall represent the status of the environment in the geographic area that surrounds the Host Vehicle at any time.  It shall consist of the following Actors, each of which represents a different type of object:
Other Nearby Vehicles
Other Road Users
Stationary Object
The status provided by each Actor shall relate to the surface of the roadway on which the Host Vehicle is moving and/or objects present in the surrounding geographic area.  It shall be possible for some of these Actors to be using the road and/or their own designated parts of the geographic area, e.g. pedestrian pavements, cycle lanes and bus lanes.
