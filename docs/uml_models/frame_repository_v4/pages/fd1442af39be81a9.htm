User Need 7.2.1.2


Description

The system shall be able to co-ordinate the emergency and rescue services once an incident has been detected, until all injured persons have reached a hospital.

Functions

Here are the functions that serve this User Need
  • 2.1.2.3 Plan Emergency Intervention
  • 2.1.2.4 Process Emergency Progress Reports
  • 2.1.2.5 Manage Incident and Emergency Data
  • 2.1.5 Provide Access and Maintain Data for Emergency
  • 2.1.9 Provide Emergency Operator Interface
