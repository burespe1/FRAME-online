Dataflow
ft.ppt-request_demand_service


Description

It contains a request from a Pre-Trip Traveller for an On-Demand Service to be created.  The Traveller shall provide details of the origin, destination, time/date Service needed, arrival time, plus any other relevant information, e.g. disabled, elderly, luggage and young children.
