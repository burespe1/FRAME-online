fo.rmo-confirm_maintenance_activities


Description

It contains either confirmation from the Road Maintenance Operator that the long or short term maintenance activity and/or equipment maintenance previously proposed by the System the can proceed, or an alternative set of maintenance activities that are to be implemented.
