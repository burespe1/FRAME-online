Low Level Function
3.1.6.1 Process Road Network Static Data

Overview

This Function shall be capable of providing the following facilities:

(1) The ability to receive road network static data from both urban and inter-urban functionality
(2) The ability to process the received data so that it can be used in the road network model by the Traffic Simulation Engine functionality.
(3) When the data has been processed it shall be sent to the functionality that manages the store of Traffic Simulation Data from where it can be obtained by the Traffic Simulation Engine functionality.

Functional Requirements

(a) continuously monitor for the arrival of the data flows containing static data for the inter-urban and/or urban road network
(b)when either of the data flows in (a) is received, process the data into a coherent set that represents the road network managed by the system, integrating the new data with any that was received previously
(c) when (b) is complete, send include the data in the network data for simulation data flow and send it to the store of Manage Traffic Simulation Data.
