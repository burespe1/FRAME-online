User Need 4.1.4.2


Description

The system shall be able to collect evidence of unauthorised access when payment has not been received.

Functions

Here are the functions that serve this User Need
  • 1.5.1 Check User's rights
  • 1.5.2 Detect Payment Violations
  • 1.5.3 Detect Access violations
  • 1.5.5 Block Access to Service
