Dataflow
tfc-pepf_account_status_request


Description

It contains a request for details of the current status of a Traveller's account that is used to purchase services. The data flow consists of the following items:
- the EP account ID
- the financial clearing-house which manages the source from which the money will be transferred
- the ID of the source account
