Dataflow
mt.pshvs_inter-urban_lane_commands


Description

It contains use of lane commands for in-vehicle display and use. This output will direct all types of Drivers to take actions so that the progress of their vehicles will make best use of the inter-urban road network. The location coded area where the command is to be displayed and where it is valid is also included.
