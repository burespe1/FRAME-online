Dataflow
pshvs_road_information_for_display


Description

It contains the current legal speed limit and/or road information for display to the Driver.  The road information will comprise changes in the road geometry and road layout for which road signs are available, e.g. sharp bends, road junctions (with or without traffic lights), changes in the number of lanes.
