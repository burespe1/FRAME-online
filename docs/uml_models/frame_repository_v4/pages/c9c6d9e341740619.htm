Low Level Function
5.12.6 Detect Illegal Use

Overview

This Function shall be capable of providing the following facilities:

(1) The ability to send a message to the relevant authority whenever a signal is received indicating that the Vehicle is not being used properly.
(2) The message shall include the Vehicle location, which shall be received from other functionality

Functional Requirements

(a) whenever the illegal use data flow is received indicating that the vehicle is being used illegally, send the stolen vehicle notification data flow to functionality in the Provide Safety and Emergency Facilities Functional Area and include in it the vehicle ID and vehicle location
(b) when as a result of (a) the send stop message data flow is received, send the stop stolen vehicle data flow to the communicate with in-vehicle systems function
(c) when after completing (a) the vehicle position for illegal use data flow is received, check if the vehicle has moved and if so, resend the stolen vehicle notification data flow to functionality in the Provide Safety and Emergency Facilities Functional Area and include in it the new vehicle location
(d) when the vehicle ID for illegal use or vehicle position for illegal use data flows are received retain their contents for use when the vehicle is being used illegally.
