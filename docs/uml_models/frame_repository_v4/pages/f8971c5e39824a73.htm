Dataflow
psle_prosecution_file


Description

It is a completed version of the fraud notification data flow. Therefore the data flow will contain a complete set of the following data elements:
- reference
- date
- type of fraud (violation)
- result of fraud (violation)
- involved user ID (if available)
- involved vehicle ID (if relevant and available)
- image of fraud (if available)
- location of fraud (violation)
- other data according to the type of fraud (violation)
