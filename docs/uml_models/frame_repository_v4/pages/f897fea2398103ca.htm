DFD 1.2 Manage User's Account

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall manage the electronic payment account used by the user.  It shall allow the user to credit his or her electronic payment account.  The Function shall debit the account according to the different transactions accomplished.  It shall also provide the user with information about the transactions that he or she has performed.

Radiating from these Functions are several Data Flows, some of which enable the Functions to load and read data from Stores.
