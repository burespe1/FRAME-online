Dataflow
pepf_credit_operator_transaction_order


Description

It defines the movement of funds to be performed between the EP account of the user, and the account of the operators. The data flow contains the following elements:
- date / time
- user ID
- service ID
- list of operators or information providers IDs
- for each operator (or information provider)
account ID
amount to be transferred
