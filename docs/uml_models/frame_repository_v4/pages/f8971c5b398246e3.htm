Dataflow
mt_urban_traffic_incident_data


Description

It contains raw traffic data from points in the urban road network from which incidents can be detected, plus current and historic traffic data for the road network, such as flows, occupancies, queues, head-ways, etc. This data will be used in detecting and managing incidents that occur in the urban road network.
