Dataflow
ft-pepf_payment


Description

  It constitutes the means by which the Traveller pays the service fee.  The means of payment may be either the selection of an account number, or that a specific payment card will be used.  The data flow includes the following elements:

- selected mode of payment (account / card)
- if use of an EP account :
- account ID
- selected mode of debiting (immediate / differed / scheduled)
