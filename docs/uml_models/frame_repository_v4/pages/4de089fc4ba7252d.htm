DFD 5.14 Provide In-vehicle Trip Planning & Implementation

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall enable a Driver to plan and implement a trip from the Vehicle.  It shall only be available in the Vehicle and shall not support the panning and/or implementation of trips that involve the use of other transport modes.  As part of its trip implementation facilities the Function shall enable trips to be revised and re-planned to accommodate changes in traffic conditions, including incidents that were not present when the trip was planned.
