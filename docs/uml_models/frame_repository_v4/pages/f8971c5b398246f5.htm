mt_urban_traffic_management_s&g_requests


Description

It contains details of the outputs that are to be displayed by stop and go (s&g) output devices to drivers and other travellers using the urban road network.  These details may include but not be limited to the identity and/or location of the actuator and the actual output to be displayed.
