DFD 5.15.3 Determine Host Vehicle Situation

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall be responsible for processing the data about the location of the Host Vehicle and its trajectory to determine if  it is being driven safely and/or it will be in conflict with any other Vehicles or Other Road Users.  If the Driver needs to take any corrective action to driver more safely and/or avoid the prospective conflict with another Vehicle or Road User, it shall be determined by the Function and sent to the Provide Vehicle Trajectory Information to Driver Function for output to the Driver.
