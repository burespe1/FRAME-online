DFD 5.13 Provide Vehicle Data and ISA

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall provide the supporting functionality for Intelligent Speed Adaption (ISA) and provides the vehicle’s current position for other functions. It also collects and then forwards probe, or floating car, data and provides speed and headway information to Drivers.
