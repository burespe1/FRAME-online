User Need 7.4.1.37


Description

Hazardous Location Notification - The system shall be able to warn driver, via an in-vehicle device, of adverse driving conditions ahead (e.g. slippery road, low visibility, queuing traffic) and advise on the appropriate action (e.g. speed).

Functions

Here are the functions that serve this User Need
  • 3.1.1.5.20 Output c&i to Drivers using Urban Roads
  • 5.12.10 Provide V2V Communications
  • 5.15.1.1 Detect Other Vehicles near to Host Vehicle
  • 5.15.1.5 Detect Visibility in Host Vehicle vicinity
  • 5.15.1.6 Detect Road Surface State in Host Vehicle vicinity
  • 5.15.4 Provide Vehicle Trajectory Information to Driver
  • 5.15.5 Collect & forward local Host Vehicle conditions

User Need Category

7.4.1 Road Hazard Warning
