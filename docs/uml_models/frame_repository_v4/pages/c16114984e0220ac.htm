Dataflow
pshvs.mt_inter-urban_slow_objects_message


Description

It contains data from which a warning message can be generated for output to the Driver in the Host Vehicle from the roadside to advise of slow moving objects (e.g. person, animal, slow vehicle, including Vulnerable Road Users (VRU's)) near by in the inter-urban road network and any appropriate action that the Driver needs to take.
