Dataflow
flea-psle_user_registration


Description

It contains the registration of a user or operator using the road network. The data flow includes the following elements:
- law enforcement agency identification
- date and place of registration
- user or operator ID
- list of vehicles for which the user or operator is responsible,
- nature of link between the vehicles and the user (owner, driver, …)
- nature of operations registered for these vehicles
- period of validity of registration.
