Dataflow
ft-trip_plan_change_approval


Description

It contains confirmation from the Traveller that the previously described changes can be made to the trip plan that is currently being implemented.  These changes will occur whilst the trip is in progress and be caused by either changes in the conditions within the travel network of a request from the Traveller.
