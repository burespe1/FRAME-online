Dataflow
pshvs.mt_urban_speed_being_exceeded


Description

It contains data that can be used to output a warning message from the roadside to the Driver of the Host Vehicle (identity included) that the Vehicle is being driven in a way that the recommended urban speed and/or legal urban speed limit is currently being exceeded.
