Dataflow
tfc-pepf_transaction


Description

It is used to ask the Financial Clearinghouse to transfer money to an account of an operator (or information provider).  The data flow includes the following elements:
- date
- financial clearing-house ID
- operator or information provider ID
- ID of the account to be credited
- ID of the EP account
- amount to be transferred
