List of Functional Areas


Definition of a Functional Area

The functionality in the System (or ITS) has been divided into eight parts called Functional Areas. These contain Functions, Data Flows and Data Stores that are closely connected to each other because their functionality is related. Thus for example, the functionality connected with the provision of assistance to Travellers is located in the same Functional Area. Each Functional Area is numbered (1 to 8) and each number provides that first digit for the number of each of the Functions and Data Stores that it contains.
