Dataflow
tt-pepf_account_loading_problem


Description

It warns the user that a problem has occurred for the crediting of the Traveller's electronic payment account. The most significant problem will be that the account managed by the Financial Clearinghouse and indicated as the source of payment is not supplied with enough financial resources, e.g. credit.
