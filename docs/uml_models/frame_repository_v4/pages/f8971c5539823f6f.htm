Dataflow
mffo_driver_request


Description

It carries a specific request from Manage fleet resources function. The request can be an instruction or an answer for additional information (i.e. a request on information that cannot be found on the resource data store) to the driver. Included in this data flow are data items such as, trip ID, driver ID, nature of the request and request.
