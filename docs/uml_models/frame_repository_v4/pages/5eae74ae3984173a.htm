Dataflow
psef.mt_inter-urban_emergency_route_request


Description

It contains the request for the implementation of priority for the use of selected lanes as part of a previously defined route for use by vehicle(s) belonging to the Emergency Services to minimise the delay that the vehicle(s) will experience as it(they) use(s) the route through the inter-urban road network.
