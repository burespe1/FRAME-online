Dataflow
mffo_commercial_request


Description

It contains data that is a specific request from Manage fleet business transaction. The request can be a commercial instruction or an additional information answer (i.e. a request on information that cannot be find on the resource data store). The data comprises trip ID, nature of the request and the request.
