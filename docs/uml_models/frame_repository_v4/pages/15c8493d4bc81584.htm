Dataflow
td.fvd-rest_area_unavailable_for_new_eta


Description

It contains an indication to the Freight Vehicle Driver that the originally booked parking space is no longer available because the Estimated Time of Arrival (ETA) of the Freight Vehicle has changed.  Details of alternative times when a suitable parking space will be available are included.
