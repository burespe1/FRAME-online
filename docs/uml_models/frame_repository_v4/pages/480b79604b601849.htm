Dataflow
mt.pshvs_global_incident_related_warnings


Description

It contains details about an incident that are to be displayed via in-Vehicle systems to Drivers using the road network. The location-coded area where the warning is to be displayed and will be valid is included, plus details of the lane(s) and road section(s) affected and the expected delay.
