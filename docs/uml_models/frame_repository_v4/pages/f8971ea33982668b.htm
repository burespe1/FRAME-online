Dataflow
psle.mffo_violation_notification


Description

It contains the following information about a violation that is being returned to the Manage Freight and Fleet Operations functionality:
- characterisation of a violation
- date
- time
- driver ID (if available)
- location of violation detection and vehicle ID.
