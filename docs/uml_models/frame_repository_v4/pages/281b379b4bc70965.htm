Dataflow
pshvs.mt_rest_area_eta


Description

It contains the Expected Time of Arrival (ETA) of the Freight Vehicle at the previously booked parking space in the rest zone that is part of a service area.  The update has been made using data produced from monitoring progress with the Vehicle Trip Plan for the Freight Vehicle that is currently being implemented.
