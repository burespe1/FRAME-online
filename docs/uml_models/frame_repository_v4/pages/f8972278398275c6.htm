Dataflow
fesp-pepf_tariff_grids


Description

It contains all the elements necessary to determine the tariff for a given service and contract.  The data flow includes the following elements:
- date,
- service provider ID,
- service ID
- for each service ID :
     - tariff for every combination of parameters defining the service, and for various contract types
