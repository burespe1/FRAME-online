Dataflow
td-current_speed_limits_and_headways


Description

It contains the output to the Driver of the Host Vehicle of the current legal limit or suggested dynamic speed, plus the suggested headway for the Vehicle.  Two values shall be included, one for the road segment currently being occupied by the Vehicle and the other for the expected next segment.  The reason(s) for any change(s) in speed indication will be included.
