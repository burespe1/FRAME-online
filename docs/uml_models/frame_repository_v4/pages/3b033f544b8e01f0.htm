Dataflow
pshvs.mt_urban_display_errors


Description

It contains either details of a difference that has been found between the commands, warnings and information plus lane and speed commands that are being displayed to the Driver from within the Host Vehicle and those that are visible to the Driver from the roadside in the urban network, or a warning that a Vehicle (whose identity is included) is exceeding the recommended and/or legal speed limits.
