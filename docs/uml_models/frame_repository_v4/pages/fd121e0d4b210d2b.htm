Dataflow
to.rno-environmental_data_analysis_results


Description

It contains the results of the analysis of current and predicted environmental data by the Determine Environmental Actions Function for display to the Road Network Operator.  The results will include one or more suggested actions that are for the Operator to confirm if they are to be implemented.
