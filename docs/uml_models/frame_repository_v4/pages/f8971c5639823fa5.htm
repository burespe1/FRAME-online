Dataflow
mffo_fleet_transport_opportunity_request


Description

  It is a request from the fleet supplier to freight system to get information about current freight opportunity.  The data flow includes fleet supplier name and address, criteria for transport opportunity selection: start time, end-time, start location, end location, cargo type.
