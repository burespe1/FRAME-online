Dataflow
fd-pepf_payment


Description

It provides the means by which the driver pays the service fee. The data may be either the selection of an account number, or the use of specific payment cards. It contains the following elements:
- selected mode of payment (account / card)
- if use of an EP account :
- account ID
- selected mode of debiting (immediate / differed / scheduled)
