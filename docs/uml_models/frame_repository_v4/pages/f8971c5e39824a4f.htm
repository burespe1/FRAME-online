Dataflow
psle_load_violations_store


Description

It contains data about a current fraud (violation) that is being sent to the Fraud Data Store. The data flow contains the following elements:
- reference
- date
- type of fraud
- result of fraud
- involved user ID (if available)
- involved vehicle ID (if relevant and available)
- image of fraud (if available)
- location of fraud
- other data according to the type of fraud
