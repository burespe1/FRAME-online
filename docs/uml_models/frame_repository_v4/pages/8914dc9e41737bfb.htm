Dataflow
tv.ov-other_platooning_data


Description

It contains the data concerned with performing platooning with one or more Vehicles that needs to be sent from the Host Vehicle to the Other Vehicle.  The identity of the Host Vehicle will be included in the data so that the Other Vehicle knows from which Vehicle in the platoon the data was sent.
