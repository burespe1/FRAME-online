Data Store
3.5 Demand Data


Description of content

This Data Store shall be used within the Manage Traffic Area.  It shall be split into two parts, one to contain data collected on the use of transport modes and the other to contain demand management strategies.

The collected data in the Store shall be structured in the following way:
- date (date string)
- time (date string)
- transport mode (characters)
- amount of use (numbers as a percentage)

The number of entries shall be fixed by the number of transport modes, the period over which they are stored

The data about the demand re-balancing strategies in the Store shall be structured in the following way:
- identity (characters)
- actions (characters).

The number of entries shall be fixed by the number of strategies.
