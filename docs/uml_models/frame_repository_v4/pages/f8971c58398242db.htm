Dataflow
mt_bridge_equipment_status


Description

It contains data that is being transferred from the bridge management functions to those for maintenance management.  The data flow contains the current status of bridge monitoring equipment and the bridge infrastructure.  This data shows whether or not the status is faulty, or that the equipment or bridge infrastructure requires maintenance.
