User Need 9.5.3.39


Description

The system shall be able to add a time stamp and/or vehicle position to any message sent from the vehicle; this data to be added at the time the message was formulated and/or the time it was sent.

Functions

Here are the functions that serve this User Need
  • 8.3.1.4 Monitor Operational Task
  • 8.3.2.2 Monitor Vehicle

User Need Category

9.5.3 Road Vehicle, Driver, Equipment & Cargo Management
