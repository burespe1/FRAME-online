Dataflow
pshvs_suggested_safe_speed_and_headway


Description

It contains the current suggested speed and headway for the Vehicle that apples to the road segment that it is using for use in monitoring the way that it is being driven so that reports are provided when it becomes unsafe, i.e. poor safety behaviour.
