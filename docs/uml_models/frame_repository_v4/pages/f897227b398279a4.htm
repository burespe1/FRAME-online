Dataflow
tcc.fs-fleet_transport_capacity_availability


Description

It indicates to freight shipper that the Fleet Supplier is available to carry freight and so to study transport opportunity. The data flow includes Fleet Supplier name and address, start time of availability, end time of availability, description of capability during availability period.
