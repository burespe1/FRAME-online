Dataflow
ft-pepf_transaction_info_request


Description

It is used to ask for the list of transactions (including account loading) performed by a Traveller. The data flow includes the following elements:
- user ID
- vehicle ID (optional)
- contract ID (optional)
- period concerned
- service ID (optional)
- EP account ID
- location of service used (optional)
- operators or information providers ID (optional)
