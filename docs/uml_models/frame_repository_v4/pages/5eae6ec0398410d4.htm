External Service Provider

External Service Provider

This is a Terminator diagram which provides a description of the Terminator breakdown structure into Actors.

Overview

This Terminator shall represent entities that interface with the System in different ways.  It shall be consist of the following Actors:
Bookable Service Provider
Broadcaster
Cellular Communications Provider
Driver and Vehicle Information Provider
Freight Storage Renting Agency
General Information Provider
Geographic Information Provider
Multi-Modal Travel Information Provider
Planned Event Organiser
Traffic and Travel Information Provider
Vehicle Renting Agency
Some of these Actors shall simply provide information to the System, e.g. map data, information about points of interest.  Other Actors shall interact with the System for the provision of services, e.g. output of traffic and travel information by broadcast radio, rental of freight storage facilities.
