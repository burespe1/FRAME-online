Terminator Other Mode Freight System
 Diagram   Definition    TabMemoryProcess

Overview

This Actor within the Multi-modal System Terminator shall represent an entity that shall provide the link to systems that are responsible for the conveyance of freight using modes of transport other than road, e.g. water, air, and rail.  These systems shall exchange data with the System to enable the synchronisation between the use of the different modes in order to maximise the efficiency of freight transport, e.g. to reduce the waiting time at modal interchanges.
