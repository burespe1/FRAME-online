Dataflow
mt_urban_speed_value


Description

It contains a speed value that is to check for violations by vehicles in a part of the urban road network. This is being implemented automatically as part of a time of day dependent sequence of changes, or at the request of the Operator, or as a result of the implementation of an incident management strategy.
