DFD 6.8 Manage Trip Plan Store

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall provide facilities for the management of the Private Trip Plan Data Store.  It shall be able to accept data from of functionality to enable the trip plan created by the Traveller to be stored.  When the Traveller requests that a previously prepared trip plan is implemented, this Function shall ensure that it is correctly provided to the trip plan implementation and monitoring functionality.
