Dataflow
mffo_request_for_fleet_resources_availability


Description

It carries the information that an analysis must be made to determine what are the best suitable transport resources for a given road freight transport operation. It also carries the information that a registered fleet transport operation must have its transport conditions changed. There are two steps in the transaction: initial where the best suitable resources are looked for ; final where the final choice is made. The data flow includes the transaction step (initial or final), transaction type (new or change), fleet transaction ID and cancelled fleet transaction ID.
