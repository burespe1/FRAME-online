Dataflow
mt_inter-urban_lane_management_requests


Description

It contains requests for a lane use restriction, keep-in-lane advice, or auxiliary lane availability to be imposed (or a current setting cleared) on one or more parts of the inter-urban road network. This change is being implemented automatically either as part of a time of day dependent sequence of changes, or as part of an incident, demand management, or environmental strategy.
