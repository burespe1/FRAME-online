Dataflow
psle_violation_notification_for_storage


Description

It contains details of an illegal action (fraud, or violation) that has been performed.  The data is provided by different Functions, and therefore may be incomplete when sent between some of them.  When complete, the data flow contains the following elements:
- reference
- date
- type of fraud or violation
- result of fraud or violation
- involved user ID (if available)
- involved vehicle ID (if relevant and available)
- image of fraud or violation (if available)
- location of fraud or violation
- other data according to the type of fraud or violation
