Dataflow
pshvs.pscs_freight_vehicle_left_un/loading_zone


Description

In contains confirmation that the Freight Vehicle has left the loading or unloading zone.  This confirmation is obtained from processing the data produced by monitoring the current implementation of the Vehicle Trip Plan for the Freight Vehicle.
