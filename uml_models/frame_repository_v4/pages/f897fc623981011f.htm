Data Store
1.7 Access Rights Data


Description of content

This Data Store shall be used within the Provide Electronic Payments Facilities Area.  It shall contain the access rights for all kinds of users, all kinds of services, and all environment conditions.  These rights shall be for each user and shall be independent of the financial conditions of the user's account.

The data in the Store shall be structured in the following way:
- service ID (number)
- user category (characters)
- environmental condition (characters)
- access rights, which shall be defined as unlimited; limited to some duration, limited to some period, etc. (characters, but may include date strings)

Note that for each service it shall be possible for there to be several user categories and environmental conditions, each with a different set of access rights.  It shall be possible for each set of access rights to consist of one or more entries.
