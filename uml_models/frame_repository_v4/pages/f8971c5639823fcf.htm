Dataflow
mffo_freight_transport_order


Description

It carries characteristics about the freight operation that will be performed. The data flow includes the fleet transaction ID (freight transaction ID and fleet supplier ID), freight shipper name and address, fleet supplier name and address, origin/destination conditions, departure/arrival dates conditions, cargo/freight characteristics, agreed price and electronic signature.
