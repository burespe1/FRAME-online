Dataflow
pshvs_current_recommended_speeds_and_headways


Description

It contains the current suggested speeds and headways being applied by ISA for display to the Driver as the recommended maximum speed and minimum headway.  Separate values shall be provided for the segment of the road network that the Vehicle is occupying and the expected next segment.  The reason(s) for any changes are to be included.
