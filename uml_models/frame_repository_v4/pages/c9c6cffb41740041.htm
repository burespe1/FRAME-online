Dataflow
mt.pshvs_urban_suggested_speeds_and_headways


Description

It contains suggested speed and headway settings for urban roads.  They are to be used by "intelligent" vehicles that are capable of controlling their speed and headway to an externally defined value.  The reason(s) for any changes are to be included.
