Dataflow
mffo_resource_incident


Description

It informs the function Manage fleet resources that an unforeseen event happens that may have consequences on resource management so that function can take all necessary decisions. The data flow contains data such as the trip ID with a brief description of the incident, and information on status for all the involved resources.
