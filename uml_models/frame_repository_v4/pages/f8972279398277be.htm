Dataflow
ft.ptt-basic_trip_parameters


Description

It contains basic data about the trip that is to be planned and may include data such as, the date on which the trip will be made (or start), the locations of the origin and destination of the trip, the departure and arrival times, places to be visited (or passed through, i.e. way points) for the trip, places to be avoided, travel modes to be used and/or avoided, the type of road Vehicle that will be used for all of part of the trip, the identity of the Traveller preparing the trip (enables their General Trip Preferences (GTP) to be used, if available), the number of Travellers and information about any goods that are being carried.
