Dataflow
pepf_transaction_information_L


Description

  It contains all the elements related to a transaction between a user and an operator (or information provider).  The data flow contains the following elements:
- date / time of payment
- date / time of service utilisation
- user ID and/or vehicle ID
- operator ID or IP ID
- location of transaction
- service ID
- other parameters defining precisely the service
- mode of payment
- amount
- user's EP account ID
