Dataflow
pscs_hazardous_routes_data_read


Description

It contains data being read from the data store of Hazardous Vehicle Routes.  This data may include the criteria used for selecting routes and/or the actual routes that have been determined for particular Vehicles that are carrying Hazardous Goods.
