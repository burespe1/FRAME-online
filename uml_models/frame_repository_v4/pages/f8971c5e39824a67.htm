Dataflow
psle_violation_type_from_compliance_check


Description

It contains the identification of the type of fraud or violation that has been enacted.  The data flow contains the following elements:
- domain of fraud or violation (payment, traffic rules violation, access violation, ...)
- type of fraud or violation within domain,
- parameters characterising the fraud or violation (difference between actual speed and authorised speed, weight in excess of max authorised weight, incorrect payment, ...)
