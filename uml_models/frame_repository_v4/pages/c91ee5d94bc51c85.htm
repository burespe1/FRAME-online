Dataflow
fesp.g-digital_map_data_for_emergency_vehicle


Description

It contains digital map data that will be used to guide the Emergency Vehicle Driver along a green wave route and/or to enable the Emergency Vehicle to know which junctions are controlled by traffic signals so that local priority can be requested if a green wave route is not being used.
