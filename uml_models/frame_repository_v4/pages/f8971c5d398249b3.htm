Dataflow
pepf_user_transaction_history


Description

It contains details about the transaction performed by the user during a certain period. The data flow contains the following elements:
- user ID or vehicle ID
- for each transaction:
- date / time
- location
- operator or information provider ID
- service ID
- parameters allowing to precisely define the service used
- ID of contract used
- account ID
- corresponding amount
