Dataflow
pepf_debit_account_order_for_user


Description

It contains data that is used to remove from the EP account the amount of the service fee.  The data flow contains the following elements:
- date of message
- user ID
- EP account ID
- amounts to be debited
- dates of debit (in case of a buy on credit)
- destination of funds
