Dataflow
mffo_answer_on_fleet_ressources_availability


Description

It carries the information that the best suitable transport resources for a given road freight transport operation have been looked for. The action can be successful or not. The data flow includes the fleet transaction ID, resources research result (positive or not) and analysis ID.
