Dataflow
mt.psle_urban_vehicle_identity_error


Description

It contains an indication that a lack of correspondence has been found between the image of a Vehicle using the urban road network, its registration data, or the identity in has included in any FCD/XFCD that it has provided.  All of the relevant data is included, such as the Vehicle image, the registration data and if available the FCD/XFCD identity.
