Dataflow
pshvs.pscs_holding_zone_eta


Description

It contains the Expected Time of Arrival (ETA) of the Freight Vehicle at the previously booked holding zone.  The update has been made using data produced from monitoring progress with the Vehicle Trip Plan for the Freight Vehicle that is currently being implemented.
