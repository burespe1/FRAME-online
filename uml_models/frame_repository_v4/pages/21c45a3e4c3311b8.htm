Dataflow
pshvs_host_vehicle_ghost_driving_warning


Description

It contains data that is to be used to output a warning message and give advice to the Driver of the Host Vehicle that the Vehicle is being driver the wrong way along the carriageway that it is occupying, i.e. it is a "ghost driver".
