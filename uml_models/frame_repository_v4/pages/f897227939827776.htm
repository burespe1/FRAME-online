Dataflow
ft.st-eCall_message


Description

It contains an e-Call input produced by a Static Traveller who is not in a Vehicle. The e-Call message must contain : time, location, involved vehicles and status, involved people and health status, and any relevant information that can be used by the Emergency Services.
