Dataflow
pepf_user_last_pass


Description

It contains information about the last recorded detection of the user (or vehicle) which is of interest for the computation of data related to infrastructure usage: occupancy rates, travel times. The data flow contains the following elements:
- user or vehicle ID
- last recorded detection's, with
- location of detection
- ID of detecting device
- time / date of detecting device
