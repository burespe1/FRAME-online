Dataflow
psef.mt_urban_emergency_route_request


Description

It contains the request for the implementation of a previously defined route for use by vehicle(s) belonging to the Emergency Services.  The request will probably involve the use of what are called "green waves" that will minimise the delay that the vehicle(s) will experience as it(they) use(s) the route through the urban road network.
