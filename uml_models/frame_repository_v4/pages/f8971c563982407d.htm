Dataflow
mffo_request_for_fleet_choice


Description

It carries the information that a fleet operator must be looked for. It also carries the information that a registered freight transport operation must have its transport conditions changed. There are two steps in the transaction: initial where a best offer is looked for ; final where the final choice is made. The data flow includes the transaction step (initial or final), transaction type (new or change), freight transaction ID (consignor order ID and freight operator proposal ID) and cancelled freight transaction ID.
