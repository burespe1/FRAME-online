DFD 2.1.2 Manage Emergency Intervention

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall include all the functionality that is focused on the processing of one emergency.  It shall receive, analyse and classify incidents in order to get a well characterised and defined emergency.  Then it shall plan the intervention to resolve the emergency situation, and start and process the intervention
