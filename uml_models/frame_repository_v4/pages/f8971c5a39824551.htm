Dataflow
mt_planned_inter-urban_data_update


Description

It contains either new or changed data that defines planned changes to traffic management strategies for the inter-urban road network, or a request for the output of this data. This data is for use by the Function that generates requests for the implementation of these planned changes in traffic management strategies.
