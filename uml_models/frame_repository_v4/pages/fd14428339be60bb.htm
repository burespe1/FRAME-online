User Need 10.4.1.1


Description

The system shall be able to provide in-vehicle general (dynamic) PT information, as well as the arrival time at, and name of, the next stop for this vehicle.

Functions

Here are the functions that serve this User Need
  • 4.1.6 Predict Vehicle Timings
  • 4.1.9 Output Arrival Information to Passengers

User Need Category

10.4.1 Information Handling
