Dataflow
ptja_other_travel_data_request


Description

It contains a request for information to be provided about the conditions and/or services provided by other modes of transport (e.g. heavy rail, air and ferry) and/or POI and/or PS information that the Travel Information Operator has found is not available in the Other Mode and POI/PS Data Store.
