Terminator Cellular Communications Provider

Overview

This Actor within the External Service Provider Terminator shall represent a provider of cellular communications that has agreed to extract data from users, from which travel times can be determined.  It will be assumed that the data that is provided will have had all personal identification removed and that it will show the journey times between defined locations.
