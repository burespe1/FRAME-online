mt.pshvs_incident_data_for_vehicle_trip_monitoring


Description

It contains information about the strategy that is being implemented in response to an incident that has been detected by the Manage Traffic functions and which may affect the Vehicle Trip Plan being implemented by a Driver using an In-vehicle device.
