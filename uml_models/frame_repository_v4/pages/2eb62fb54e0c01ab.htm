Dataflow
pshvs_vehicle_data_for_attitude_determination


Description

It contains data such as speed and trajectory from the Vehicle System in the Host Vehicle that is for use in determining that the Host Vehicle is in the proper attitude, i.e. it has not fallen over, or rolled upside down.
