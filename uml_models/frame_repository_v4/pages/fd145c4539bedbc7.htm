Dataflow
psef.pshvs_eCall_first_acknowledgement


Description

It contains an initial acknowledgement of an "eCall" that has been received from a Vehicle.  This type of acknowledgement would be used to indicate to the Vehicle occupants that their "eCall" has been received and is being processed.
