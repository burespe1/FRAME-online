Dataflow
mffo_request_for_freight_optimization


Description

It carries the information that a study must be made to determine whether or not a road freight transport operation can be optimised with respect to inter-urban and multi-modal transport. Or it can request to inform about a registered road freight transport operation. The data flow includes the freight transaction ID (consignor order ID and freight operator proposal ID) and request type (study or information).
