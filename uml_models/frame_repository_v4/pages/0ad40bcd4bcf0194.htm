DFD 9.4 Provide Hazardous Goods Management

This is a DFD diagram which provides a description of the function in the ITS environment

Overview

This High-level Function shall enable the routes used by Freight Vehicles carrying Hazardous Goods to be planned and for the use of these routes to be monitored.  Drivers of these Vehicles shall be sent route guidance instructions so that they can follow the prepared route.
