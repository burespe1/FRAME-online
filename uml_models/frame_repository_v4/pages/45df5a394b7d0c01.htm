Dataflow
ptja_requested_copy_of_GTP_data


Description

It contains the requested copy of the data that is currently in the GTP Data Store for output to the Travel Information Operator.  It will have all Traveller identities, payment details and other personal data removed so that the Preferences cannot be in any way associated with a particular Traveller or a group of Travellers.
