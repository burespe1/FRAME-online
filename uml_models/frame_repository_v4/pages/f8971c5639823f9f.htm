Dataflow
mffo_fleet_transport_offer


Description

It carries characteristics about the conditions under which the freight operation could be performed. The data flow includes the fleet transaction ID (fleet supplier proposal ID and freight transaction ID), fleet supplier name and address, departure/arrival dates proposal, proposed price and electronic signature.
