Dataflow
psle_fraud_characteristics


Description

It regroups all the elements making it possible to establish the prosecution file. The data is based on the fraud (violation) notification, to which are added the following:
- any image, record available, with description or ID of recording means
- classification of fraud (violation)
- other offences references enacted by the same person
