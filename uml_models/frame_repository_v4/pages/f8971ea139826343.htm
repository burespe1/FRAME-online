Dataflow
mffo.psle_violation_notification


Description

It contains all the elements gathered by Functions in the Manage Freight and Fleet Operations Area about a violation affecting its operations.  It is composed of several data flows: date, image of violation (if available), involved driver / operator ID (if available), involved vehicle ID (if relevant and available), location of violation, other data according to the type of violation, reference, result of violation and type of violation.
