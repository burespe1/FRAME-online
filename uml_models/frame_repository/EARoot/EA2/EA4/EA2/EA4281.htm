This Terminator is part of the Operator Terminator and shall represent a human entity that uses the functionality in the system to manage the traffic using the road network and the operation of the functionality that provides the management.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent commands, data and requests for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The functionality in the system must also be able to communicate with more than one human entity that is a Road Network Operator, each of which may belong to the same organisation or to different organisations and which may be responsible for different sections of the road network.<br/><br/>Mnemonic: o.rno<br/>
