This Terminator shall represent external entities that interface with the functionality in the system in different ways.  Each entity shall be represented by one of the following Terminators:<br/><br/>Bookable Service Provider<br/>Broadcaster<br/>Driver and Vehicle Information Provider<br/>Freight Storage Renting Agency<br/>General Information Provider<br/>Geographic Information Provider<br/>Mobile Communications Provider<br/>Multi-modal Travel Information Provider<br/>Planned Event Organiser<br/>Traffic and Travel Information Provider<br/>Vehicle Renting Agency<br/><br/>Some of these Terminators shall provide data objects to the system either on request from functionality in the system, or whenever a change occurs to the previously provided data objects.  They are expected to contain information that is relevant to the system operation and the way it provides ITS Services.  Other of these Terminators shall provide data objects that are requests for functionality in the system to respond with data objects containing information about its operation.<br/><br/>Mnemonic: esp<br/>
