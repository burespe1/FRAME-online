This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which it shall be possible to ask the Traveller or Driver for payment for the selected service.<br/>(2) Once the Traveller or Driver has responded to the request for payment, the ability to check with Credit Control for the transaction to be validated.<br/>(3) If the transaction is not valid for some reason the ability to register it as an incorrect payment (fraud).<br/>
