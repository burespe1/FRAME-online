Details:
(a) continuously monitor for receipt of the user detection and/or vehicle detection data flows from the Detect User function, to indicate that a Vehicle or Traveller has been detected;<br/>(b) when either of the data flows in (a) is received, ask for identification from the Traveller or the Driver using the ID request data flows;<br/>(c) if a Vehicle has been detected, a request for its identity is also sent to the Provide Vehicle ID function in the vehicle ID request data flow;<br/>(d) when the identity has been received in either (b) or (c), a record of the service use is created in the store of Transaction Data (D1.4) using the pass record data flow.<br/>(d) the identity is sent to the Check User's Rights function in either the user ID for rights check data flow or the vehicle data for right check data flow;;<br/>(e) the last related transaction for this user is read from the store of Transaction Data (D1.4) and used to calculate infrastructure usage information;<br/>(f) the information produced in (e) is sent to the urban traffic, inter-urban traffic and demand management functionality in the Manage Traffic Functional Area.<br/>
